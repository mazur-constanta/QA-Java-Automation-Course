- 3 cостояния файлов репозитория: 1.   Файлы просто созданы и всё 2. Гит следит за определенными файлам (попадают в индекс) 3. Создание контрольной точки, кормит
- git add -A
- git status (Already tracked in green)
- git commit -a -m"Checked data types, basics"
- git log  - изменения кем и когда
- git remote add origin https://github.com/constancegerman/JS_basics.git
- git push -u origin master

Merge работа
git branch (проверка, что мы на главной ветке)
git branch -D lesson_2 (для удаления уже ненужной ветки)
git branch lesson_3 (создаем новую ветку)
git switch lesson_3 (переключаемся на новую ветку локальную)
git branch (проверка, что мы на новой ветке)
git status  - проверить статус (Untracked in red)
git add L3_Theory_homework/ (добавили новый в staged)
git status  - проверить статус (Tracked in green)
git commit -m"added theory lesson 3" (добавляем коммит)
git switch main (возвращаемся на главную локальную ветку)
git branch (проверка, что мы на главной ветке)
git push origin lesson_3 (отправляем изменения на удаленный репазиторий)
